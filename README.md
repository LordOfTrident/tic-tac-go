## Introduction
I just started learning Go, so to get more familiar with it i wrote my first game in it - Tic Tac Go

## Quickstart
```sh
$ make
$ ./bin/app
```

## Controls
| Key              | Action                                          |
| ---------------- | ----------------------------------------------- |
| Left Mouse Click | Place X or O/restart game when the game is over |
| ESC              | Quit                                            |
